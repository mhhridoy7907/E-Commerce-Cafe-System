# ☕ Coffee E-Commerce System (Google Sheets Backend)

A lightweight full-stack e-commerce web application for selling coffee products using **Google Sheets as a database** and **Google Apps Script as a backend API layer**.

---

## 🚀 Features

- 📦 Product listing (coffee items with price, image, stock)
- 🛒 Add to cart system
- 💳 Checkout flow (customer details + order placement)
- 📊 Order storage in Google Sheets
- 🧾 Simple admin-level data management via Sheets
- 📱 Responsive frontend UI

---

## 🧱 Tech Stack

- **Frontend:** HTML, CSS, JavaScript  
- **Backend:** Google Apps Script  
- **Database:** Google Sheets (lightweight DB)  
- **API Type:** REST-like (GET/POST via Apps Script Web App)
